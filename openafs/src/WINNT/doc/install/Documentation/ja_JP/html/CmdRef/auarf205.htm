package apropos

目的

キーワード文字列を含む各ヘルプ項目を表示します。

構文 

package apropos [-topic <help string>]  [-help]

package a [-t <help string>]  [-h]

構文

package apropos コマンドは、 package コマンド (名前または簡単な説明に、 -topic 引き数が指定する文字列を含みます) の、オンライン・ヘルプ項目の最初の行を表示します。

コマンドの構文を表示するためには、 package help コマンドを使用してください。

オプション

-topic
検索するキーワード文字列を、英小文字のみで指定します。その文字列が 2 語以上である場合は、それを、二重引用符 ("") あるいはその他の区切り文字で囲んでください。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

コマンドのオンライン・ヘルプ項目の最初の行には、コマンドの名前と簡単な機能説明があります。このコマンドを実行すると、 package コマンドの最初の行が表示されます。 -topic 引き数で指定された文字列は、このコマンド名または最初の行に含まれています。

例

次のコマンドで、package コマンドのうち、その名前または簡単な説明に、 help という語を含むものがすべてリストされます。 

   % package apropos help
   apropos: search by help text
   help: get help on commands

必要となる特権

なし。
