salvage.lock

目的

区画に対して複数のサルベージ操作を同時に実行しないようにします。

構文

salvage.lock ファイルは、ファイル・サーバー・マシンで実行されるサルベージャー (salvager プロセス) を一度に 1 つだけにします (単一プロセスを複数のサブプロセスに分割すると、複数の区画を並列でサルベージできます)。サルベージャーは初期化時に、空のファイル (ゼロ長) を /usr/afs/local ディレクトリーに作成し、flock システム・コールを呼び出します。このファイルは、サルベージ操作が完了すると削除されます。サルベージャーはこのファイルをロックして実行しなければならないので、一度に 1 つのサルベージャーしか実行できません。
