管理解説書

Vvol_ID.vol

目的

AFS ボリュームを記述します。

構文

Vvol_ID.vol ファイルは、ボリューム ID vol_ID を持つ AFS ボリュームのヘッダー・ファイルです。 AFS サーバー (/vicep) の区画には、このようなファイルが 1 つ、ボリュームごとに保存されています。ヘッダー・ファイルには、ボリューム名、ID 番号、タイプ (読み取り / 書き込み、読み取り専用、またはバックアップ)、サイズ、および状況 (オンライン、オフライン、または使用中) などの情報が記録されています。ヘッダー・ファイルの情報を表示する場合には、vos listvol または vos examine コマンドを使用してください。

ヘッダー・ファイルは、ボリューム内のデータを参照していますが、実際のデータはありません。 AFS ファイル・スペースにボリュームをマウントし、キャッシュ・マネージャーを介して内容を表示する場合を除いて、AFS データにアクセスすることはできません。
