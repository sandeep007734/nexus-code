uss apropos

目的

キーワード文字列を含むヘルプ項目を表示します。

構文 

uss apropos -topic <help string>  [-help]

uss ap -t <help string>  [-h]

構文

uss apropos コマンドは、いずれかの uss コマンドのオンライン・ヘルプ項目の最初の 1 行を表示します。 -topic 引き数で、コマンド名および概要に含まれる文字列を指定します。

コマンドの構文を表示するには、uss help コマンドを使用します。

オプション

-topic
検索するキーワード文字列を、英小文字のみで指定します。複数の単語を入力する場合には、各単語を二重引用符 ("") か、他の区切り文字で囲んでください。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

コマンドのオンライン・ヘルプ項目の最初の行には、コマンドの名前と簡単な機能説明があります。このコマンドを使用すると、 -topic 引き数で指定した文字列がコマンド名または最初の行に含まれている uss コマンドが表示されます。

例

次のコマンドは、名前または説明部分に create という言葉が含まれている uss コマンドをすべて表示します。 

   % uss apropos create
   add: create a new user

必要となる特権

なし。
