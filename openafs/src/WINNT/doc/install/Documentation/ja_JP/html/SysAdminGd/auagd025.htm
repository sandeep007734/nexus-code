付録 D. AIX 監査イベント

この付録では、AIX ファイル・サーバー・マシン上で監査できる AFS イベントの完全なリストを提供します。AIX ファイル・サーバー・マシン上の AFS イベントの監査についての指示は、AFS パフォーマンスのモニターおよび監査 の章を参照してください。

概要

以下のリストは、ファイル /afs/usr/local/audit/events.sample に含まれる AFS イベントのリストです。各項目には、イベント・クラス、イベント名、イベントに関連したパラメーター、およびイベントの説明に関する情報が含まれています。

大部分のイベントは、関連したエラー・コードを持っていて、イベントの結果 (各イベントは、発生後記録される)、AFSName (要求するプロセスの認証識別)、およびホスト ID (要求の発生元) を表示します。イベントの多くは、 AFS Programmer's Reference Manual に定義されている RPC サーバー項目のコールに従います。

イベントは、機能別に分類されています (これは、AIX 固有です)。イベントのなかには、ファイル /usr/afs/local/config.sample で定義される以下の 1 つ以上のクラスに分類されるものもあります。

  • A (afsauthent): 認証および識別イベント
  • S (afssecurity): セキュリティー・イベント
  • P (afsprivilege): 特権必須イベント
  • O (afsobjects): オブジェクト作成および削除イベント
  • M (afsattributes): 属性変更
  • C (afsprocess): プロセス制御イベント

イベント クラス パラメーター 説明
AFS_Audit_WR なし <string> ファイル "/usr/afs/Audit" が、書き込まれました (AIX 固有のイベント)。

イベント クラス パラメーター 説明
AFS_Aud_NoCall S ECode Event このイベントを持つ rx call 構造はありません。セキュリティー、AFS ID または call のオリジンを取得できません。
AFS_Aud_NoConn S ECode Event rx call に関連した接続情報はありません。セキュリティー、AFS ID または call のオリジンを取得できません。
