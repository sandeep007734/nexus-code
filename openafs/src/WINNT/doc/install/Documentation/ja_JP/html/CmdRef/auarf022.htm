FORCESALVAGE

目的

区画全体を強制的にサルベージします。

構文

FORCESALVAGE ファイルが、 (/vicep ディレクトリーの) AFS サーバー区画に存在する場合、サルベージャーは、区画全体をサルベージしなければならないというシグナルを送られます。 AFS 修正バージョンの fsck プログラムを使用すると、区画に破壊が検出されたときに空 (0 長) のファイルが生成されます。サルベージャーは、サルベージ操作が完了すると削除されます。

ファイル・サーバーは、ボリュームを装着している区画にこのファイルを検出すると、停止して、すでに装着されているボリュームをすべてはずし、メッセージを /usr/afs/logs/FileLog ファイルに記録してから終了します。次に Bos サーバーは、サルベージャーを呼び出し、この区画をサルベージします。
