package_test

目的

パッケージ構成ファイルの妥当性をテストします。

構文 

package_test <config file>

このコマンドは、AFS コマンド・セットの構文規則に準拠していません。完全なコマンド名を指定してください。

構文

package_test コマンドは、プロトタイプ・ファイルがコンパイルされるときに作成される、 パッケージ構成ファイルの妥当性をテストします。コマンド・インタープリターは、標準出力ストリーム上にエラー・メッセージを印刷します。

オプション

config file
検証するパッケージ構成ファイルを指定します。

例

次の例では、パッケージ構成ファイル、 staff.sun4x_56 の妥当性をテストしています。 

   % package_test staff.sun4x_56

必要となる特権

なし。
