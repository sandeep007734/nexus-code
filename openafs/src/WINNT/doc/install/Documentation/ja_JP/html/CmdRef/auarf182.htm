kas apropos

目的

キーワード文字列を含む各ヘルプ項目を表示します。

構文 

kas apropos -topic <help string>  [-help]

kas a  -t <help string>  [-h]

構文

kas apropos コマンドは、 -topic 引き数で指定された文字列と、名前または簡単な説明が一致する kas コマンドの、オンライン・ヘルプの最初の行を表示します。

コマンドの構文を表示するには、kas help コマンドを使用します。

オプション

-topic
検索するキーワード文字列を、英小文字のみで指定します。複数の単語を入力する場合には、各単語を二重引用符 ("") か、他の区切り文字で囲んでください。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

コマンドのオンライン・ヘルプ項目の最初の行には、コマンドの名前と簡単な機能説明があります。このコマンドを実行すると、 -topic 引き数に指定された文字列がコマンド名または最初の行にある kas コマンドの最初の行が表示されます。

例

次のコマンドは、名前または説明部分に key という言葉が含まれている kas コマンドをすべて表示します。 

   % kas apropos key
   setkey: set a user's key
   stringtokey: convert a string to a key

必要となる特権

なし。パスワードは不要です。
