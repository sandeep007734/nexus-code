fstrace apropos

目的

キーワード文字列を含む各ヘルプ項目を表示します。

構文 

fstrace apropos -topic <help string>  [-help]

fstrace ap -t <help string>  [-h]

構文

fstrace apropos コマンドは、 fstrace コマンドの名前または簡単な説明の中で、 -topic 引き数に指定された文字列を含むコマンドのオンライン・ヘルプの最初の行を表示します。

コマンドの完全な構文を表示するには、fstrace help コマンドを使用します。

オプション

-topic
検索するキーワード文字列を、英小文字のみで指定します。複数の単語を入力する場合には、各単語を二重引用符 ("") か、他の区切り文字で囲んでください。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

コマンドのオンライン・ヘルプ項目の最初の行には、コマンドの名前と簡単な機能説明があります。このコマンドを実行すると、 -topic 引き数に指定された文字列がコマンド名または最初の行にある fstrace コマンドの最初の行が表示されます。

例

次のコマンドは、名前または説明部分に set という言葉が含まれている fstrace コマンドをすべて表示します。 

   % fstrace apropos set
   clear: clear logs by logname or by event set
   lsset: list available event sets
   setlog: set the size of a log
   setset: set state of event sets

必要となる特権

なし。
