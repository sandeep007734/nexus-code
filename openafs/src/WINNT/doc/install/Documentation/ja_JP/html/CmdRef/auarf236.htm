translate_et

目的

番号化されたエラー・コードをテキスト・メッセージへ変換します。

構文 

translate_et  <error number>+

このコマンドは、AFS コマンド・セットの構文規則に準拠していません。完全なコマンド名を指定してください。

構文

translate_et コマンドは、指定されたエラー番号をテキスト・メッセージへ変換します。

オプション

error number
変換するエラー番号を指定します。

例

以下のコマンドは、エラー番号 1 および 4 を変換します。 

   % translate_et 1 4
   1 ().1 = Not owner
   4 ().4 = Interrupted system call

必要となる特権

なし。
