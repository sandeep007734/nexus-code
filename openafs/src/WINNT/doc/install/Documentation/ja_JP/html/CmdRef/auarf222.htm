管理解説書

pts membership

目的

ユーザーまたはグループのメンバーシップ・リストを表示します。

構文 

pts membership -nameorid <user or group name or id>+  [-cell <cell name>]
               [-noauth]  [-force]  [-help]

pts m -na <user or group name or id>+  [-c <cell name>]  [-no]  [-f]  [-h]

pts groups -na <user or group name or id>+  [-c <cell name>]
           [-no] [-f]  [-h]

pts g -na <user or group name or id>+  [-c <cell name>]  [-no]  [-f]  [-h]

構文

pts membership コマンドは、 -nameorid 引き数に指定された、ユーザーまたはマシンが属するグループをリスト表示します。あるいは、 -nameorid 引き数に指定された、各グループに属するユーザーまたはマシンをリスト表示します。

system:anyuser のメンバーまたは、 system:authuser グループを、リスト表示することはできません。またこれらは、ユーザーが属するグループのリストには表示されません。

ユーザーまたはマシンをグループに追加するには、 pts adduser コマンドを使用してください。それらを削除する場合は、 pts removeuser コマンドを使用してください。

オプション

-nameorid
ユーザー項目の場合には、グループ・メンバーシップを表示するユーザー項目の名前または AFS UID を、マシン項目の場合には、マシンの IP アドレス (完全名、またはワイルドカードを使用した形式) または AFS UID、さらにグループの名前または AFS GIDを指定してください。同じコマンド行に、ユーザー、マシンおよびグループに加えて、名前と ID も混在させることができます。負であることを示すために、各グループの GID の前にハイフンを付けてください。

-cell
コマンドを実行するセルを指定します。詳細については、pts の概要に関する解説ページを参照してください。

-noauth
発行者に、非特権 ID anonymous を割り当てます。詳細については、 pts の概要に関する解説ページを参照してください。

-force
このフラグを使用すると、エラーや他の問題が発生したときに、最初のエラーでコマンドの実行を停止するのではなく、可能な限り継続します。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

ユーザーおよびマシンごとに、出力は次のようなヘッダー行で始まり、ユーザーやマシンが属するグループのリストがそれに続きます。 

   Groups name (id: AFS UID) is a member of:

グループごとに、出力は次のようなヘッダー行で始まり、グループに属するユーザーやマシンのリストがそれに続きます。 

   Members of group_name (id: AFS GID) are:

例

次の例では、ユーザー pat が属するグループと、グループ smith:friends のメンバーをリスト表示しています。項目 pat の 3 番目の privacy フラグは、デフォルトのハイフンが変更され、非管理ユーザーがこのリストを取得できるようになっているので注意してください。 

   % pts membership pat smith:friends
   Groups pat (id: 1144) is a member of:
     smith:friends
     staff
     johnson:project-team
   Members of smith:friends (id: -562) are:
     pat
     terry
     jones
     richard
     thompson

必要となる特権

-nameorid 引き数に指定された、各ユーザーまたはグループの保護データベース項目の、 3 番目の privacy フラグの設定によって、必要な特権が異なります (このフラグを表示するには、 pts examine コマンドを使用します)。

  • それがハイフンで、 -nameorid 引き数にユーザーが指定されている場合には、関連したユーザーと、 system:administrators グループのメンバーだけが、ユーザーの属するグループをリスト表示することができます。
  • それがハイフンで、 -nameorid 引き数にマシンが指定されている場合には、 system:administrators グループのメンバーだけが、マシンの属するグループをリスト表示することができます。
  • それがハイフンで、 -nameorid 引き数にグループが指定されている場合には、グループの所有者と、 system:administrators グループのメンバーだけが、グループのメンバーをリスト表示することができます。
  • 小文字の m で、-nameorid 引き数に、ユーザーまたはマシン項目が指定されている場合には、ハイフンの場合と同じ意味になります。
  • 小文字の m で、 -nameorid 引き数にグループが指定されている場合には、グループ・メンバーが他のメンバーをリスト表示することもできます。
  • 大文字の M の場合には、セルのデータベース・サーバー・マシンにアクセスできればだれでも、グループ・メンバーシップをリスト表示することができます。
