backup help

目的

指定された backup コマンドの構文、またはすべての backup コマンドの簡単な説明を表示します。

構文 

backup help  [-topic <help string>+]  [-help]

backup h [-t <help string>+]  [-h]

構文

backup help コマンドは、 -topic 引き数によって指定された各操作コードについての、完全なオンライン・ヘルプ項目 (概説および構文) を表示します。 -topic 引き数が省略されると、各 backup コマンドのオンライン・ヘルプ項目の最初の行 (名前と簡単な説明) を表示します。

名前または説明部分に特定のキーワードを含む backup コマンドを表示する場合には、 backup apropos コマンドを使用してください。

オプション

-topic
各コマンドの完全なオンライン・ヘルプ項目が表示されます。操作コードだけを指定する場合には、コマンド名から backup を除いた部分を指定してください。 (たとえば、backup dump ではなく、 dump と指定します。) この引き数を省略すると、各 backup コマンドの簡単な説明が表示されます。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

各 backup コマンドのオンライン・ヘルプ項目は、次のように 2、3 行から構成されます。

  • 最初の行には、コマンド名および機能の説明が表示されます。
  • 次の行には、コマンドの別名 (ある場合) が表示されます。
  • 最後の行は、Usage という文字列で始まり、コマンドのオプションが一定の順序で表示されます。オンライン・ヘルプの項目では、この文書の解説ページと同じ記号 (大括弧など) が使用されています。

例

次のコマンドは、 backup dump コマンドのオンライン・ヘルプ項目を表示します。 

   % backup help dump
   backup dump: start dump
   Usage: backup dump -volumeset <volume set name> -dump <dump level name>
   [-portoffset <TC port offset>]  [-at <Date/time to start dump>+]
   [-append]  [-n]  [-file <load file>]  [-help]

必要となる特権

なし。
