package help

目的

指定された package コマンドの構文を表示するか、すべての package コマンドの機能説明をリストします。

構文 

package help [-topic <help string>+]  [-help]

package h [-t <help string>+]  [-h]

構文

package help コマンドは、 -topic 引き数が指定する個々のコマンド操作コードに対する、完全なオンライン・ヘルプ項目 (簡単な説明および構文ステートメント) を表示します。 -topic 引き数が省略される場合は、個々の package コマンドの、オンライン・ヘルプ項目の最初の行 (名前と簡単な説明) が出力されます。

名前または簡単な説明に特定のキーワードが含まれる、 package コマンドをリストするには、 package apropos コマンドを使用してください。

オプション

-topic
各コマンドの完全なオンライン・ヘルプ項目が表示されます。操作コードだけを指定する場合には、コマンド名の package の部分を省略してください (たとえば、 package initcmd ではなく、 initcmd と指定します)。この引き数を省略すると、各 package コマンドの簡単な説明が出力されます。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

個々の package コマンドのオンライン・ヘルプ項目は、次のような 2 行または 3 行で構成されます。

  • 最初の行には、コマンド名および機能の説明が表示されます。
  • 次の行には、コマンドの別名 (ある場合) が表示されます。
  • 最後の行は、Usage という文字列で始まり、コマンドのオプションが一定の順序で表示されます。オンライン・ヘルプの項目では、この文書の解説ページと同じ記号 (大括弧など) が使用されています。

例

次のコマンドは、package initcmd コマンドの、オンライン・ヘルプ項目を表示します。 

   % package help initcmd
   package initcmd: initialize the program
   Usage: package [initcmd] [-config <base name of configuration file>]
   [-fullconfig <full name of configuration file, or stdin for standard input>]
   [-overwrite] [-noaction] [-verbose] [-silent] [-rebootfiles]
   [-debug] [-help]

必要となる特権

なし。
