管理解説書

backup delvolset

目的

バックアップ・データベースから 1 つ以上のボリューム・セットを削除します。

構文 

backup delvolset -name <volume set name>+
                 [-localauth]  [-cell <cell name>]  [-help]

backup delvols -n <volume set name>+  [-l]  [-c <cell name>]  [-h]

構文

backup delvolset コマンドは、 -name 引き数で指定されたボリューム・セットと、そのボリューム項目をバックアップ・データベースから削除します。 backup listvolsets コマンドを使用すると、現在バックアップ・データベースに定義されているボリューム・セットとそのボリューム項目をリストすることができます。

注意

一時ボリューム・セットの削除は、それが作成された対話式セッション内でのみ行うことができます。対話式セッションを終了した場合にも、一時ボリューム・セットは自動的に破棄されます。

オプション

-name
削除するボリューム・セットの名前を指定します。

-localauth
ローカルの /usr/afs/etc/KeyFile ファイルのキーを使用して、サーバー・チケットを構成します。相互認証の間、 backup コマンド・インタープリターは、バックアップ・サーバー、ボリューム・サーバーおよび VL サーバーにこのチケットを提供します。このフラグは、-cell 引き数と一緒に使用してはなりません。詳細については、backup の解説ページを参照してください。

-cell
コマンドを実行するセルを指定します。この引き数は、 -localauth フラグと一緒に使用してはなりません。詳細については、backup の解説ページを参照してください。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

例

次のコマンドは、user というボリューム・セットと、そこに含まれるすべてのボリューム項目を削除します。 

   % backup delvolset user

必要となる特権

コマンドの発行者は、バックアップ・サーバーが実行されているすべてのマシン上の /usr/afs/etc/UserList ファイルにリストされていなければなりません。また、-localauth フラグを使用する場合は、ローカル・スーパーユーザー root としてサーバー・マシンにログオンする必要があります。
