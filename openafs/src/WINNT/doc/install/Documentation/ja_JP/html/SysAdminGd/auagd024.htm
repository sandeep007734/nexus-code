付録 C. afsmonitor プログラムの統計

この付録は、afsmonitor プログラムにより得られた統計の一覧です。統計は、カテゴリーおよびセクション別に収集して、各フィールド、グループ、およびセクションが簡単に説明されています。 afsmonitor プログラムの使用方法については、AFS パフォーマンスのモニターおよび監査 を参照してください。

キャッシュ・マネージャー統計

キャッシュ・マネージャー統計フィールドは、以下の機能グループおよびグループに分類されています。

  • PerfStats_section - パフォーマンス統計機能グループ
    • PerfStats_group - パフォーマンス統計グループ
    • misc_group - 各種グループ
  • Server_UpDown_section - サーバー Up/Down 統計機能グループ
    • FS_upDown_SC_group - 同一セル・グループ内のファイル・サーバー Up/Down 統計
    • FS_upDown_OC_group - 別のセル・グループ内のファイル・サーバー Up/Down 統計
    • VL_upDown_SC_group - 同一セル・グループ内の VL サーバー Up/Down 統計
    • VL_upDown_OC_group - 別のセル・グループ内の VL サーバー Up/Down 統計
  • RPCop_section - RPC オペレーション測定機能グループ
    • FS_RPCopTimes_group - ファイル・サーバー RPC オペレーション・タイミング・グループ
    • FS_RPCopErrors_group - ファイル・サーバー RPC オペレーション・エラー・グループ
    • FS_RPCopBytes_group - ファイル・サーバー RPC 転送タイミング・グループ
    • CM_RPCopTimes_group - キャッシュ・マネージャー RPC オペレーション・タイミング・グループ
  • Auth_Access_section - 認証および複写済みファイル・アクセス・機能グループ
    • Auth_Stats_group - キャッシュ・マネージャー・グループ用認証情報
    • Access_Stats_group - 未複写ファイル・アクセス・グループ

これらの機能グループ名およびグループ名で分類される全キャッシュ・マネージャー変数は、以下にリストします。

パフォーマンス統計機能グループ (PerfStats_section)

パフォーマンス統計グループ (PerfStats_group)

  • dlocalAccesses: ローカル・セル内のファイルへのデータ・アクセス数
  • vlocalAccesses: ローカル・セル内のファイルへの stat アクセス数
  • dremoteAccesses: ローカル・セル外のファイルへのデータ・アクセス数
  • vremoteAccesses: ローカル・セル外のファイルへの stat アクセス数
  • cacheNumEntries: キャッシュ項目数
  • cacheBlocksTotal: キャッシュ用に構成された (1K) ブロックの数
  • cacheBlocksInUse: 現行使用中のキャッシュ・ブロックの数
  • cacheBlocksOrig: ブート時のキャッシュ・ブロックの数
  • cacheMaxDirtyChunks: ダーティー・キャッシュ・チャンクの許容最大数
  • cacheCurrDirtyChunks: ダーティー・キャッシュ・チャンクの現行数
  • dcacheHits: ローカル・キャッシュで検索されるデータ・ファイル数
  • vcacheHits: ローカル・キャッシュで検索される stat 項目数
  • dcacheMisses: ローカル・キャッシュで検索されないデータ・ファイル数
  • vcacheMisses: ローカル・キャッシュで検索されない stat 項目数
  • cacheFlushes: キャッシュからフラッシュされるファイル数
  • cacheFilesReused: 再利用されるキャッシュ・ファイル数
  • ProtServerAddr: 使用される保護サーバーのアドレス
  • vcacheXAllocs: 追加で割り振られる vcaches
  • bufAlloced: AFS によって割り振られるバッファー数
  • bufHits: バッファー・キャッシュ上で検索されるページ数
  • bufMisses: バッファー・キャッシュ上で検索されないページ数
  • bufFlushDirty: すべて使用中のためフラッシュされたキャッシュ済みダーティー・バッファーの数
  • LargeBlocksActive: 現在使用済みのラージ空きプール項目の数
  • LargeBlocksAlloced: 割り振り済みのラージ空きプール項目の数
  • SmallBlocksActive: 現在使用済みのスモール空きプール項目の数
  • SmallBlocksAlloced: 割り振り済み使用済みのスモール空きプール項目の数
  • OutStandingMemUsage: 割り振り済みメモリー容量
  • OutStandingAllocs: 未解決の osi_allocs (未だ osi_frees でない)
  • CallBackAlloced: 割り振り済みコールバック構造の数
  • CallBackFlushes: 実行済みコールバック・フラッシュ・オペレーションの数
  • srvRecords: 現在レコード上にあるサーバー数
  • srvRecordsHWM: サーバー・レコードの上限基準点
  • srvNumBuckets: サーバー・ハッシュ・チェーン・バケットの数
  • srvMaxChainLength: サーバー・ハッシュ・チェーンの最大長
  • srvMaxChainLengthHWM: サーバー・ハッシュ・チェーンの上限基準点
  • sysName_ID: ホスト・ハードウェアの Sysname ID

各種グループ (misc_group)

  • numPerfCalls: 受信済みパフォーマンス・コール数
  • epoch: キャッシュ・マネージャーのエコー時間
  • numCellsVisible: 認識済みのセル数
  • numCellsContacted: アクセス済みのセル数
