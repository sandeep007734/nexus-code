fstrace help

目的

指定された fstrace コマンドの構文、またはすべての fstrace コマンドの簡単な説明を表示します。

構文 

fstrace help [-topic <help string>+]  [-help]

fstrace h [-t <help string>+]  [-h]

構文

fstrace help は、 -topic 引き数で指定されたコマンド操作コードについて、完全なオンライン・ヘルプ項目 (簡単な説明と構文情報) を表示します。-topic 引き数が省略されると、各 fstrace コマンドのオンライン・ヘルプ項目の最初の行 (名前と簡単な説明) を表示します。

名前または説明部分に特定のキーワードを含む fstrace コマンドを表示する場合には、 fstrace apropos コマンドを使用してください。

オプション

-topic
各コマンドの完全なオンライン・ヘルプ項目が表示されます。操作コードだけを指定する場合には、コマンド名から fstrace を除いた部分を指定してください。(たとえば、fstrace clear ではなく、 clear と指定します。) この引き数を省略すると、各 fstrace コマンドの簡単な説明が表示されます。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

各 fstrace コマンドのオンライン・ヘルプ項目は、2、3 行で構成されます。

  • 最初の行には、コマンド名および機能の説明が表示されます。
  • 次の行には、コマンドの別名 (ある場合) が表示されます。
  • 最後の行は、Usage という文字列で始まり、コマンドのオプションが一定の順序で表示されます。オンライン・ヘルプの項目では、この文書の解説ページと同じ記号 (大括弧など) が使用されています。

例

以下のコマンドは、fstrace setset コマンドのオンライン・ヘルプ項目を表示します。 

   % fstrace help -topic setset
   fstrace setset: set state of event sets
   Usage: fstrace setset [-set <set_name>+] [-active] [-inactive]
   [-dormant] [-help]

必要となる特権

なし。
