管理解説書

関連資料

以下のドキュメントは、AFS のドキュメント・セットに含まれています。

AFS 管理の手引き

この手引きでは、AFS セルの管理についてシステム管理者が知っておかなければならない概念と手順が解説されています。本書の読者は UNIX に関する知識を持っていることを想定していますが、AFS の予備知識は必要ではありません。

AFS 管理の手引き の最初の章では、基本的な概念と指針について解説しています。AFS セルの管理を正しく行うには、この基本概念と指針を理解しておくことが大切です。この手引きの残りの章では、特定の管理作業について、それを実行するのに重要な概念を説明しながら、1 つ 1 つ解説しています。

AFS インストールの手引き

この手引きでは、AFS サーバーおよびクライアント・マシンの導入方法を解説しています。導入担当者は、経験を積んだ UNIX (R) システム管理者を想定しています。

予測可能なパフォーマンスを得るには、マシンはこの手引きの指示通りに導入され、構成されなければなりません。

AFS リリース情報

この文書には、新しく加わった機能とコマンドのリスト、必要な環境と制限のリスト、サーバー・マシンとクライアント・マシンをアップグレードするための手順など、AFS の各リリースに固有の情報が含まれています。

AFS 使用者の手引き

この資料は、AFS の基本的な概念、効果的に使用する上で必要な手順について説明しています。本書は UNIX を使用した経験がある読者を対象にしていますが、ネットワークまたは AFS の知識は必要ありません。

この資料には、パスワードの変更、AFS データの保護、グループの作成、トラブルシューティングなど、基本的な機能に関する説明が含まれています。また、各機能の説明ではイラストや図を利用し、UNIX ファイル・システムと AFS の違いが分かるように解説しています。
