fstrace setset

目的

イベント・セットの状況を設定します。

構文 

fstrace setset [-set <set_name>+]  [-active]  [-inactive]  [-dormant]  [-help]

fs set [-s <set_name>+]  [-a]  [-i]  [-d]  [-h]

構文

fstrace setset コマンドは、トレース・メッセージがカーネル・メモリーのログ・バッファーにレコードされたかどうかを判別するローカル・マシン上の cm カーネル・イベント設定の状況を設定します。

オプション

-set
状況を設定するイベント・セットに名前を付けます。唯一の許容値で、この引き数が省略されている場合のデフォルトも cm です。

-active
イベント設定のためのトレースを使用可能にして、関連したトレース・ログ・バッファーのためのカーネル・メモリーを割り振ります。 -inactive フラグ、または -dormant フラグの 1 つを与えます。

-inactive
イベント設定のためのトレースを一時的に使用不可能にするが、関連したトレース・ログ・バッファーのためのカーネル・メモリーの割り振りを変更しません。-active フラグ、または -dormant フラグの 1 つを与えます。

-dormant
イベント設定のためのトレースを使用不可能にして、関連したトレース・ログ・バッファーのために以前割り振られたカーネル・メモリーを空けます。-active フラグ、または -inactive フラグの 1 つを与えます。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

例

以下の例は、cm イベント設定の状況を inactive を設定します。 

   # fstrace setset -set cm -inactive

必要となる特権

コマンドの発行者は、ローカル・スーパーユーザー root としてログインする必要があります。
