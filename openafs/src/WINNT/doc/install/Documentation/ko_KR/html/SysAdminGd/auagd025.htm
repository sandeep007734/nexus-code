부록 D. AIX 감사 이벤트

이 부록은 AIX 파일 서버 기계에서 감사할 수 있는 AFS 이벤트의 완전한 목록을 제공합니다. AIX 파일 서버 기계에서 AFS 이벤트를 감사하는 방법에 대해 AFS 성능 모니터 및 감사를 참조하십시오.

소개

다음은 /afs/usr/local/audit/events.sample 파일에 들어 있는 AFS 이벤트의 목록입니다. 각 항목에는 이벤트 클래스, 이벤트 이름, 이벤트에 연관된 매개변수 그리고 이벤트 설명에 대한 정보가 들어 있습니다.

대부분의 이벤트에는 이벤트의 결과를 보여주는 연관된 오류 코드(각 이벤트는 발생된 다음에 기록되므로), AFSName(요청하는 프로세스의 인증 ID) 그리고 호스트 ID(요청이 시작된 것)가 있습니다. 많은 이벤트는 AFS Programmer's Reference Manual에서 정의된 RPC 서버 항목 다음에 옵니다.

이벤트는 기능별로 나누어집니다(이는 AIX 고유). 일부 이벤트는 /usr/afs/local/config.sample 파일에서 정의된 다음 클래스 중 두 개 이상에 속할 수도 있습니다.

  • A (afsauthent): 인증 및 식별 이벤트
  • S (afssecurity): 보안 이벤트
  • P (afsprivilege): 특권이 필요한 이벤트
  • O (afsobjects): 오브젝트 작성 및 삭제 이벤트
  • M (afsattributes): 속성 수정
  • C (afsprocess): 프로세스 제어 이벤트

감사-고유 이벤트


이벤트 클래스 매개변수 설명
AFS_Audit_WR 없음 <string> "/usr/afs/Audit" 파일은 쓰여졌습니다(AIX 고유 이벤트)
AFS_Aud_On S ECode 이 서버 프로세스에서 감사가 켜졌습니다(서버가 시동될 때 기록됨).
AFS_Aud_Off S ECode 이 서버 프로세스에서 감사가 꺼졌습니다(서버가 시동될 때 기록됨).
AFS_Aud_Unauth S ECode Event 허가 받지 않은 사용자에 의해 이벤트가 시작되었습니다.
주:다음 감사-고유 이벤트는 이벤트를 기록할 때 발생한 오류를 나타냅니다. 대부분의 이벤트에는 연관된 AFSName과 호스트 ID가 있습니다. 이 정보를 Rx 구조 밖에서 모을 수 없으면 이런 이벤트 중 하나가 발생합니다.

이벤트 클래스 매개변수 설명
AFS_Aud_NoCall S ECode Event 이 이벤트에는 rx 호출 구조가 없습니다. 보안, AFS ID 또는 호출 기점을 확보할 수 없습니다.
AFS_Aud_NoConn S ECode Event rx 호출과 연관된 연경 정보가 없습니다. 보안, AFS ID 또는 호출 기점을 확보할 수 없습니다.
AFS_Aud_UnknSec S ECode Event 호출의 보안을 알 수 없습니다(허가 받았거나 허가 받지 않은 호출자이어야 함).
AFS_Aud_NoAFSId S ECode Event 보안 아벤트와 연관된 AFS ID/이름이 없습니다.
AFS_Aud_NoHost S ECode Event 호출자의 기점(기계)에 대한 정보가 없습니다.
AFS_Aud_EINVAL 없음 이벤트 감사 이벤트 매개변수에 오류가 없습니다(이벤트 매개변수를 기록할 수 없음).
