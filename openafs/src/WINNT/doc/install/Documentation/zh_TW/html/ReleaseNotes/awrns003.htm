簡介

概觀

AFS(R) 是一種分散式檔案系統， 對整個企業的資訊，提供安全可靠的存取方式。AFS 將個別檔案伺服器機器上的目錄與檔案緊密地整合成一個可從任何桌面上存取的檔案系統，呈現給使用者一個無機器界限的單一檔案空間。

AFS 具備許多優點。AFS 運用用戶端快取功能， 並對於經常存取的資料，提供跨越多重檔案伺服器的複製功能，來改進檔案的可用性。 AFS 提供功能強大的安全機制，保護檔案空間中儲存的資訊。 此外，AFS 具高度的調整性。 可以視需要增加幾乎毫無限制的伺服器與用戶端機器到 AFS 架構中，對於現有的伺服器與用戶端機器沒有什麼影響。 這可讓檔案系統 隨著企業一同成長。

AFS for Windows 包含下列產品：

  • AFS 伺服器

    AFS 伺服器在 Windows NT(R) 機器上執行 AFS 伺服器程序。 Windows NT 機器上的「AFS 伺服器」可架構成下列一或多種型式： 「AFS 檔案伺服器」、「AFS 資料庫伺服器」、「AFS 備份伺服器」、以及「AFS 系統控制伺服器」。AFS 伺服器提供「AFS 伺服器架構精靈」來協助設定。

  • AFS 控制中心

    AFS 控制中心提供兩個功能強大的圖形使用者介面（GUI）工具， 以協助 AFS 系統管理者執行 AFS Cell 管理：「AFS 伺服器管理程式」和「AFS 帳戶管理程式」。前者協助管理一或多個 AFS 伺服器 （在 Windows NT 和 UNIX 系統上）的容體和服務，後者可容易地建立和維護 AFS 使用者與群組帳戶。

  • AFS 用戶端

    AFS 用戶端可從執行 Windows NT 的 PC 上直接存取 AFS 檔案空間，讓使用者能管理 AFS 的檔案和目錄。

  • AFS Light

    AFS Light 可藉由轉送 AFS 要求到已安裝 AFS 用戶端的 PC 上， 並從執行 Windows 98 或 Windows 95 的 PC 上存取 AFS 檔案空間。
