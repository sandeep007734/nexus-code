Welcome to the Network Identity Manager OpenAFS Provider

The OpenAFS Provider extends the Network Identity Manager (or NetIDMgr for short) to support AFS credential management. This Provider obtains AFS tokens for the cells associated with each identity and provides the tools necessary to maintain them.

This version of the provider is distributed by OpenAFS as part of OpenAFS for Windows.  Visit http://www.openafs.org/windows.html for updates.
